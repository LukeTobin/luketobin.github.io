Notable Work

Level Design

The goal with creating levels for this prototype was to take design principles from the original treyarch zombie maps. Utilizing sections of interest & open area which players will spend most of their time

The map gets split into tight areas leading to your most common area to stay. A more fleshed out prototype would include points of interest within the tight areas, giving the player a reason to leave the core area.

Zone System

A design method I brough over to my prototype was to utilize the same system structure that Treyarch used in the zone system

This system splits areas of the map into unique zones. This has a series of strong benefits for the game. It causing the spawning of zombies to be closer to the player rather than random areas far away from the player, it also allows for better recyling of distant enemies.

Zombies

The zombie's of this game utilize a series of different useful systems. Zombies movement is based on the Unity NavMesh, which allows for optimal pathfinding towards the character. With a constant flow of zombies, the system I created utilized the object pooling design pattern, to recycle objects instead of the heavy use of creating & destroying objects.

Weapon System

The weapon system was set up to be modular and allow for easy customization, possible of handling variation in many forms such as single fire, burst, automatic and delayed. Each weapon can edit their settings easily for quick customization.

Variables such as weapon sway, aim down sight time and recoil spread are examples of unique editable variables.

Custom Shaders

The prototype utilizes an interesting shader effect, which is a combination of an outline shader & a toon shader. This shader effect applied upon a low poly model created an intriguing aesthetics for the game, similar to that of borderlands.

Due to the nature of how the shader was set up, it was possible to apply this shader effect on any existing material and mix well with it. This allowed for me to apply the shader on all objects in the scene, allowing for that awesome borderlands look across the game.

Multiplayer System

The game includes a multiplayer system. This was set up using Photon PUN, the game itself includes a working lobby system & capability to play with others. Features such as lobby name, size & custom usernames are all also available.
