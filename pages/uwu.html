uwu Machine

The concept of the game is to generate funds to repair a broken machine by selling different types of plants. The game was created for the Global Game Jam 2020

The game was developed alongside 1 other developer over 72 hours and was made entirely within Unity canvas. My contribution included creating:

  • Growing Plants
  • UI Functionality
  • Selling Plants
  • Harvesting Tools
  • Game Loop Logic
  • Upgrades
